# 🌸 Skinora Cosmetic Care Website

Skinora Cosmetic Care is a responsive and user-friendly cosmetic care website developed using HTML, CSS, and JavaScript.  
The project provides beauty and skincare related information with an attractive interface and interactive features.

---

## ✨ Features

- 🧴 Product Information Section
- 🌿 Ingredients Details
- 💄 Skin Care Routine Guide
- 📊 Product Comparison Page
- 📝 Feedback Form
- ❓ Beauty Quiz Section
- 📞 Contact Page
- 📱 Responsive Web Design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

---

## 📂 Project Pages

- Home Page
- About Page
- Products Page
- Ingredients Page
- Compare Products Page
- Routine Page
- Quiz Page
- Feedback Page
- Contact Page

---

## 🚀 How to Run the Project

1. Download or clone the repository
2. Open the project folder
3. Run `index.html` in any web browser

---

## 📸 Project Purpose

The purpose of this project is to provide users with cosmetic care information, skincare guidance, and product-related details through a simple and interactive website.
